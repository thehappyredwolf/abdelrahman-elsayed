# Abdelrahman Elsayed - Personal Website & Blog

A personal website and blog built with [Zaggonaut](https://zaggonaut.dev), a retro-inspired theme for Astro.

## About

This is my personal website where I share my thoughts, showcase my projects, and write about topics I'm interested in.

## Tech Stack

- [Astro](https://astro.build) - Web framework
- [TypeScript](https://www.typescriptlang.org) - Type safety
- [TailwindCSS](https://tailwindcss.com) - Styling
- [Zaggonaut](https://github.com/RATIU5/zaggonaut) - Theme template

## Getting Started (Local Development)

```bash
# Install dependencies
pnpm install

# Start the dev server
pnpm run dev

# Build for production
pnpm run build
```

## Content Structure

- **Blog posts** - Located in `content/blogs/`
- **Projects** - Located in `content/projects/`
- **Site config** - `content/configuration.toml`
