# Abdelrahman Elsayed

Welcome to my online presence. Here, I showcase my projects, experiences, and blogs.

## Features

- Built with Astro, Svelte, and Tailwind CSS
- Responsive design
- Light and dark mode
- Search with Pagefind
- Extended Markdown features
- Table of contents for posts
- RSS feed
- Multi-language support

## Tech Stack

- Astro
- Svelte
- Tailwind CSS
- Pagefind
- Expressive Code
- Vercel Analytics

## Getting Started

1. Clone the repository:
    `git clone <repo-url>`
    `cd <repo-folder>`
2. Install dependencies:
    `pnpm install`
3. Start the development server:
    `pnpm dev`
4. Create new posts:
    `pnpm new-post <filename>`
5. Edit your site settings in `src/config.ts`.

## Deployment

You can deploy this site to Vercel, Netlify, or GitHub Pages. Edit astro.config.mjs for site configuration.
